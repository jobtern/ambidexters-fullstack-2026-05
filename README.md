## Getting started

**Do not fork or clone this repository.**

Open it in Codespaces:

1. Click the green **Code** button above
2. Select the **Codespaces** tab
3. Click **Create codespace on main**
4. Wait 2–3 minutes for the environment to set up
5. Create a new branch using your GitHub username: `git checkout -b your-github-username`
6. Open [TASK.md](./TASK.md) — that is your brief

Your environment is pre-configured. Everything you need is already installed.

## Submitting

When you are ready to submit:

1. Make sure all your work is inside the `solution/` directory
2. Commit and push your branch
3. Open a pull request against `main`
4. Fill in the pull request template
5. Add the `ready-for-review` label — this triggers your automated review

Your review will be posted to the pull request within a few minutes.

## Rules

- All your work must be inside `solution/` — do not modify anything outside it
- You have up to 3 submission attempts — each resubmission carries a score penalty
- Do not remove or reassign labels
- Stop your Codespace when taking breaks to preserve your free usage quota

Good luck.
